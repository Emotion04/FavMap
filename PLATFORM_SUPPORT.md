# FavMap 平台支持说明

## Web 版本

### 已支持功能

- ✅ **收藏列表显示** - 显示所有收藏的地点
- ✅ **深色模式** - 支持浅色/深色主题切换
- ✅ **设置页面** - 可以配置 API Key
- ✅ **编辑收藏** - 可以编辑收藏信息
- ✅ **导航栏** - 底部标签页导航

### 需要配置的功能

- ⚠️ **地点搜索** - 需要配置高德地图 API Key
- ⚠️ **批量导入** - 需要配置高德地图 API Key
- ⚠️ **地铁站信息** - 需要配置高德地图 API Key

### 不支持的功能

- ❌ **地图显示** - react-native-maps 不支持 Web
- ❌ **定位功能** - Web 版本不支持获取位置

---

## 移动版本 (iOS/Android)

### 已支持功能

- ✅ **地图显示** - 使用高德地图 SDK
- ✅ **地点搜索** - 搜索并收藏地点
- ✅ **收藏管理** - 添加、编辑、删除收藏
- ✅ **定位功能** - 获取当前位置
- ✅ **地铁站信息** - 显示附近地铁站
- ✅ **批量导入** - 批量导入地点
- ✅ **社交分享** - 分享到社交平台
- ✅ **深色模式** - 支持浅色/深色主题
- ✅ **导航功能** - 调用地图 App 导航

### 需要配置

- ⚠️ **高德地图 API Key** - 在设置页面配置

---

## 如何配置 API Key

### 1. 获取高德地图 API Key

1. 访问 [高德开放平台](https://lbs.amap.com/)
2. 注册账号
3. 创建应用
4. 添加 Key（选择 Web 服务）

### 2. 在应用中配置

1. 打开应用
2. 点击底部"设置"标签
3. 找到"高德地图 API Key"输入框
4. 填入你的 API Key
5. 点击"保存"

## 技术限制

### Web 版本限制

1. **地图显示** - react-native-maps 不支持 Web，需要使用其他方案
2. **定位功能** - Web 浏览器需要 HTTPS 才能获取位置
3. **原生功能** - 部分原生功能在 Web 上不可用

### 解决方案

对于 Web 版本的地图显示，可以考虑：
1. 使用 Google Maps JavaScript API
2. 使用 Leaflet + OpenStreetMap
3. 使用高德地图 JS API
